# MyTool System Scanner

MyTool System Scanner is a lightweight Windows tool that displays basic system information such as CPU details, RAM usage, and operating system info. It is designed to be simple, fast, and easy to use.

## Features

- CPU information (name, cores, threads)
- RAM information (installed, available, usage)
- Operating system details (Windows version, build, architecture)
- Clean and simple interface

## Installation

Download the installer from the Releases section:

**MyToolSetup.exe**

Run the installer and follow the steps.

## Running From Source

If you want to run the Python version:

1. Install Python 3
2. Install dependencies:
   pip install psutil
3. Run the program:
   python MyTool.py

## Files in This Project

- **MyTool.py** — main program

- **MyTool.script** — script logic
